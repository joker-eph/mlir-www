Detailed Description

Wrapper around an MlirBlock.

Note that blocks can exist in a detached state (where this instance is responsible for clearing) or an attached state (where its owner is responsible).

This python wrapper retains a redundant reference to its creating context in order to facilitate checking that parts of the operation hierarchy are only assembled from the same context.
